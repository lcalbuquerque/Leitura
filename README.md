# Leitura Project

Projeto contém as seguintes funcionalidades:

1 - Criar, alterar e excluir "Posts" contendo as seguintes informações:
    - Título
	- Autor
	- Corpo (mensagem)
	- Data é criada automaticamente no instante da gravação do post

2 - Criar, alterar e excluir comentários que ficam "ligados" ao Post previamente criado

3 - Tanto para o Post como para o Comentário, o usuário pode votar positivamente ou negativamente além de poder ordená-los 
    pelos critérios de Data e Votos em ambos os casos a classificação aceita ordem ascendente e descendente.

# Como instalar

1. Vá para a pasta "api-server" e execute:
   - npm install
   - npm start. Obs: Mantenha esta janela ativa (aberta ou minimizada)
2. Em outra janela do CMD e na pasta principal do projeto, execute:
   - npm install
   - npm start
